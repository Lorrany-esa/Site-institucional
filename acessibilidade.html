Portal de Acessibilidade

Prefeitura Municipal de São João da Ponte - MG

Contraste e Leitura

O site da prefeitura utiliza cores com contraste adequado para facilitar a leitura de pessoas com baixa visão ou daltonismo.

Você pode aumentar ou diminuir o tamanho do texto utilizando a função do seu navegador (Ctrl + + / Ctrl + - no Windows ou Cmd + + / Cmd + - no Mac).

Navegação por Teclado

O site é navegável utilizando apenas o teclado. Utilize a tecla Tab para avançar pelos links e elementos interativos, e Shift + Tab para voltar.

Leitores de Tela

O site segue padrões de acessibilidade compatíveis com leitores de tela, fornecendo descrições alternativas para imagens, botões e links importantes.

Atalhos e Recursos

  • Pressione Alt + 0 para acessar o menu principal (dependendo do navegador).
  • Use a função de aumento de fonte do navegador para melhorar a leitura.
  • Os formulários possuem labels visíveis e claros para facilitar o preenchimento.
